Version 4.0

Following advice from bodo_ori , I was finally able to make this design asymmetric.
The model now has color changes, bigger bones on the right wing and a sword although the grid only got slightly bigger, which shows how much more efficient it is now (or rather how inefficient the previous version was).
I like how it turned out, though there are some small things I want to change which I only noticed when I was almost done folding and I didn't want to start again.
The collapse was a bit of a pain because the paper was too soft and not really good to work with but I wanted to have this cool pattern.

    Created July 2020
    Paper 48 cm square of double tissue
    Time ~9 hours
    Grid 64x64
